    • sm.debugDraw.addAabb(name, min, max, color?)

    Adds a named aabb debug draw.

    Parameters:

    TypeNameDescription
    stringnameThe debug transform name.
    Vec3minThe box minimum extent.
    Vec3maxThe box maximum extent.
    Colorcolor?The color. Defaults to white.

    sm.debugDraw.addArrow(name, from, to?, color?)

    Adds a named arrow debug draw.

    Parameters:

    TypeNameDescription
    stringnameThe debug arrow name.
    Vec3fromThe from position.
    Vec3to?The to position. Defaults to the from position plus one along the z axis. (World up vector)
    Colorcolor?The color. Defaults to white.

    sm.debugDraw.addBox(name, min, max, position, rotation?, color?)

    Adds a named box debug draw.

    Parameters:

    TypeNameDescription
    stringnameThe debug transform name.
    Vec3minThe box minimum extent.
    Vec3maxThe box maximum extent.
    Vec3positionThe box position.
    Quatrotation?The box rotation. Defaults to identity.
    Colorcolor?The color. Defaults to white.

    sm.debugDraw.addSphere(name, center, radius?, color?)

    Adds a named sphere debug draw.

    Parameters:

    TypeNameDescription
    stringnameThe debug sphere name.
    Vec3centerThe sphere center.
    numberradius?The sphere radius. Defaults to 0.125.
    Colorcolor?The color. Defaults to white.

    sm.debugDraw.addText(name, text, position, fade?, scale?, color?)

    Adds text in world debug draw.

    Parameters:

    TypeNameDescription
    stringnameThe text name.
    stringtextThe text content.
    Vec3positionThe world position of the text.
    booleanfade?If the text should fade. Defaults to false.
    numberscale?The scale of the text. Defaults to 30.
    Colorcolor?The color. Defaults to white.

    sm.debugDraw.addTransform(name, origin, rotation, scale?)

    Adds a named transform debug draw.

    Parameters:

    TypeNameDescription
    stringnameThe debug transform name.
    Vec3originThe transform origin.
    QuatrotationThe transform rotation.
    numberscale?The transform scale. Defaults to 1.0.

    sm.debugDraw.clear(name?)

    Removes all debug draws beginning with a given name.

    Parameters:

    TypeNameDescription
    stringname?The name to match (Defaults to "", matching all debug draws).

    Parameters:

    Colorcolor?The color of the arrow. Defaults to white.
