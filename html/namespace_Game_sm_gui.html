    sm.gui.computeTextSize(text, font, align, maxWidth?, replaceTags?)

    Client only

    Calculates the 720 widget size required to display a text

    Parameters:

    TypeNameDescription
    stringtextThe text to compute.
    stringfontThe font to use.
    stringalignThe text alignment.
    numbermaxWidth?The maximum desired widget width before wrapping (optional) (Defaults to -1)
    booleanreplaceTags?Should tags be replaced before calculating? (optional) (Defaults to true)

    Returns:

    TypeDescription
    {number, number}The calculated width and height.
