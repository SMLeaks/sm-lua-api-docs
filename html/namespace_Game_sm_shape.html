sm.shape

Associated object type: Shape

A shape is any block, part or basic material that can be built by a player. Shapes are always connected to a body, which is a collection of shapes.

For more information about creating your own scripted shapes, see ShapeClass.

    sm.shape.createBlock(uuid, size, position, rotation?, dynamic?, forceSpawn?)

    Server only

    Create a new block

    Parameters:

    TypeNameDescription
    UuiduuidThe uuid of the shape.
    Vec3sizeThe size of the block.
    Vec3positionThe shape's world position.
    Quatrotation?The shape's world rotation. Defaults to no rotation (Optional)
    booleandynamic?Set true if the shape is dynamic or false if the shape is static. Defaults to true (Optional)
    booleanforceSpawn?Set true to force spawn the shape even if it will cause collision. Defaults to true (Optional)

    Returns:

    TypeDescription
    Shape The created block

    sm.shape.createPart(uuid, position, rotation, dynamic?, forceSpawn?, world?)

    Server only

    Parameters:

    TypeNameDescription

    Vec3positionThe shape's world position.
